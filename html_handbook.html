МІЙ ЕЛЕКТРОННИЙ ДОВІДНИК HTML
ПРИЗНАЧЕННЯ НАЗВА ТЕГУ КОРОТКИЙ ОПИС
Для каркасного документа <html> Тег повідомляє браузеру, що це HTML-документ.
<head> Тег є контейнером для всіх головних елементів.
<meta> Тег містить метадані про документ HTML. Метадані не відображатимуться на сторінці, але стануть доступними для машин.
<body> Тег визначає тіло документа.
<!DOCTYPE> Декларація повинна бути найпершою справою у вашому документі HTML перед тегом .
Для роботи з текстом <p> Тег визначає абзац.
<b> Тег вказує жирний текст.
<strong> Тег фраз. Він визначає важливий текст.
<i> Тег визначає частину тексту альтернативним голосом або настроєм. Вміст тегу зазвичай відображається курсивом.
<u> Тег представляє якийсь текст, який повинен стилістично відрізнятися від звичайного тексту, наприклад, з неправильно написаними словами або власними іменами на китайській мові.
<sub> Тег визначає текст підкреслення. Текст підскладу відображається наполовину символом нижче звичайної лінії, а іноді відображається у меншому шрифті. Індексний текст може бути використаний для хімічних формул, таких як H2O.
<sup> Тег визначає текст верхнього індексу. Текст над текстом відображається наполовину символом вище звичайної лінії, а іноді відображається у меншому шрифті. Текст з написом може бути використаний для виносок, наприклад WWW [1].
<br> Створює перехід на новий рядок.
Для гіперпосилань <a> Тег визначає гіперпосилання, яка використовується для посилання з однієї сторінки на іншу.
